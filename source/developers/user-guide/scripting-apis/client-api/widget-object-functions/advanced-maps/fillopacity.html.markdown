Description

How you can set the fill opacity for overlays

## Example

    app.w('googleMapsAdvanced').fillOpacity(0.3 + 0.7 * Math.random());  //Set Random Opacity
    app.w('googleMapsAdvanced').fillOpacity(0.7);
