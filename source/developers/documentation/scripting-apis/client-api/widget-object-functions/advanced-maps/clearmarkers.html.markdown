**clearMarkers** ()

## Description

Where you can clear all markers added by [addMarker()](/developers/documentation/scripting-apis/client-api/widget-object-functions/advanced-maps/addmarker)

## Example

    app.w('googleMapsAdvanced').clearMarkers();
