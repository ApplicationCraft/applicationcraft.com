## Description

How you can set the line opacity for overlays

## Example

     
    app.w('googleMapsAdvanced').strokeOpacity(0.3 + 0.7 * Math.random());  //Set Random Opacity
    app.w('googleMapsAdvanced').strokeOpacity(0.7);
