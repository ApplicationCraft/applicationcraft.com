**play** ()

## Description

Play the specified video/audio.

## Example

     
    app.w("Video").play();
    
For jPlayer widget

	app.w("jPlayer").play(30);  //play from 30 sec allows you to Jump to a point, in seconds, from the start of the video/audio with jPlayer.
