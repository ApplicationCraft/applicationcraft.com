**language** (strLang);

## Parameters

<table>
<tr>
<td width="170">
{string} strLang

</td>
<td width="1">
</td>
<td width="710">
The language code to which to switch the App

</td>
</tr>
</table>

## Description

Switches the display language of the App to the language code specified in strLang.

## Example

      strLangCode = 'de';  
      strUserMessage = app.language(strLang);
