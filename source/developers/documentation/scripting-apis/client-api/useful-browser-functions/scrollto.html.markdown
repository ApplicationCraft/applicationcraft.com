**scrollTo** (xNum, yNum)

**scrollBy** (xNum, yNum)

## Parameters

<table>

<td width="193">
{integer} xNum, yNum

</td>
<td width="17">
</td>

X and Y coordinates to scroll by

</td>
</tr>

## Description

Scrolls the browser contents by the specified number of pixels.

## Example

    scrollBy(0, 100);  // Scrolls 100 pixels lower
