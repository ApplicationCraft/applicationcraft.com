**networkTimeout** (milliseconds)

## Parameters

<table>
<tr>
<td width="186">
{int} milliseconds

</td>
<td width="11">
</td>
<td width="683">
number of milliseconds until a timeout occurs

</td>
</tr>
</table>

## Returns

Nothing.

## Description

This specified the timeout period for Web Service calls made from SSJ.

## Example

    // Set timeout to 30 seconds
    ssj.networkTimeout(20000);
