Application Craft is one of the very best ways available to teach students about programming. It is great for letting them build Apps that run on Desktops, Tablets and Smartphones. For student developers, it also runs out-of-the-box with nothing to install whatsoever.

Students can see results really quickly, but it doesn't limit their ability to take on more and more advanced challenges.

Your servers or in the Cloud

We deliver the system as an Ubuntu Virtual Machine image in Virtual Box or VM-Ware format. We also offer an Ubuntu package.

A great alternative is to run on Amazon's EC2. We supply Amazon AMIs so you can be up, running and teaching in 5 minutes.

What you need to know

  • AC Education - Teaching Edition costs just $999 per year including almost daily updates. There are no limits on developer seats. The only limitation is that it can be used for teaching purposes only.
  • AC Education - Full Edition costs $1,999 per year and you can use it to build and run administrative Apps as well as for teaching purposes.
  • We run a special education referral program allowing you to get up to 100% off subsequent licenses.
  • If you stop paying, you can carry on using it but you don't get any updates.
  • Look for our dedicated AC Education Forum for specialized support from the commnunity, also closely monitored by us.
  • The license is the same as the Enterprise License, but with some extra bits to accommodate the above differences.
